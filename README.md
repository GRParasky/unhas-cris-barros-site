# Unhas Cris Barros — Nail Salon Landing Page

Professional landing page for a nail salon specialising in gel nails, nail art, and hand spa treatments in Penha, SC.

Built as a **freelance project** focused on local search visibility — driving organic traffic from Google and converting visitors into booked appointments via WhatsApp.

**[→ View Live Site](https://grparasky.github.io/unhas-cris-barros-site/)**

---

## Stack

- **React** — component-based, config-driven UI
- **Vite** — build tooling and dev server
- **CSS Modules** — scoped styles driven by a theme config object
- **GitHub Pages** — static hosting with automatic HTTPS

---

## Sections

- **Hero** — high-impact visual with a prominent booking call-to-action
- **Services** — clear breakdown of offered services (gel nails, nail art, hand spa, manicure)
- **Portfolio** — curated showcase of recent work
- **Contact & Booking** — WhatsApp integration and MaApp booking link

---

## Local SEO

Built from the ground up with local search as a first-class requirement:

- `BeautySalon` schema markup (JSON-LD)
- Consistent NAP (Name, Address, Phone)
- Geo meta tags targeting Penha, SC and surrounding cities
- Keywords matched to local search intent
